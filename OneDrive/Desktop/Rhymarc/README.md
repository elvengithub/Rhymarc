# Rhymarc Angelo Cabrera - Home Care Physical Therapy Website

A professional, modern, and mobile-responsive website for Rhymarc Angelo Cabrera, a licensed Physical Therapist offering specialized home care rehabilitation and fitness coaching in Cebu City and Minglanilla.

## 🚀 Features

- **Responsive Design:** Fully optimized for desktops, tablets, and mobile devices.
- **Service Pages:** Detailed information on Musculoskeletal, Neurologic, Pediatric, and Geriatric rehabilitation.
- **Booking System:** User-friendly appointment request form with integrated rates and safety policies.
- **Process Guide:** A step-by-step "How it Works" guide for new patients.
- **FAQ Section:** Answers to common questions regarding referrals, costs, and session details.
- **Direct Contact:** Quick links for phone, email, and Facebook Messenger.

## 🛠️ Tech Stack

- **Framework:** [React](https://reactjs.org/) (with TypeScript)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Routing:** [React Router Dom](https://reactrouter.com/)
- **Styling:** Custom Vanilla CSS (found in `src/styles/global.css`)

## 📋 Prerequisites

Before you begin, ensure you have the following installed:
- [Node.js](https://nodejs.org/) (v16 or higher recommended)
- [npm](https://www.npmjs.com/) (usually comes with Node.js)

## 🏃 How to Run the Project

1. **Clone the Repository (if applicable):**
   ```bash
   # If you have git installed
   git clone <repository-url>
   cd Rhymarc
   ```

2. **Install Dependencies:**
   ```bash
   npm install
   ```

3. **Start the Development Server:**
   ```bash
   npm run dev
   ```

4. **View the Website:**
   Open your browser and navigate to the URL shown in your terminal (typically `http://localhost:5173`).

## 🏗️ Project Structure

- `src/components/`: Reusable UI elements like `Navbar` and `Footer`.
- `src/pages/`: Individual page components (Home, About, Services, etc.).
- `src/styles/`: Global CSS variables and base styles.
- `src/App.tsx`: Main application shell and route configuration.

## 📝 Operating Instructions

- **Updating Content:** To change text or services, navigate to the relevant file in `src/pages/` and edit the React components.
- **Styling Changes:** Global colors, fonts, and spacing can be adjusted in `src/styles/global.css`.
- **Building for Production:** To create an optimized production build, run `npm run build`. The output will be in the `dist/` folder.
